# JavaScript-Password-Generator
Password generator made in JavaScript

# Usage
You must have Node.JS installed to use this tool, if you dont, then download Node.JS! (https://nodejs.org/en/download/).
After you download Node.JS, run the **setup.bat** file, after you do that, run the **start.bat** file and begin using the tool!
***NOTE*** If you would like to change the configuration of the tool, go into the **src** folder and edit **config.json** from true to false or vice versa.
